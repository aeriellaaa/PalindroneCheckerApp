Objective

The PalindromeCheckerApp is a simple Java console application that checks whether a hardcoded string is a palindrome and displays the result.
Project Flow

Program starts execution from the main() method

A hardcoded string is defined

The string is reversed using a loop

Original and reversed strings are compared

Palindrome result is printed

Program exits

 Running Procedure
javac UseCase2PalindromeCheckerApp.java
java UseCase2PalindromeCheckerApp
